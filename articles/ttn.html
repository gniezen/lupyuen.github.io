                    What is The Things Network?

                    The Things Network is a crowd-sourced wireless network. And it works worldwide!

                    Our IoT Devices may connect to The Things Network and transmit Sensor Data to the Cloud.

                    How much does it cost?

                    Nothing! The public community network is Free for Fair Use.

                    (The network has been free since its launch in 2015)

                    Totally free! What‚Äôs the catch?

                    Here‚Äôs what we need‚Ä¶

                    1. LoRa Wireless Module: We‚Äôll use the Semtech SX1232 LoRa Transceiver (Transmitter + Receiver) that‚Äôs bundled with our PineDio Stack Board.

                      The Things Network Global Coverage

                    3. Fair Use: Because it‚Äôs a free network for Sensor Data, we can‚Äôt spam it with messages.

                      Each device may transmit roughly 10 tiny messages per hour.

                    1.1 Configure Gateway

                    Take Note: This is missing from the RAKwireless docs‚Ä¶

                    The Things Network has been upgraded recently and there‚Äôs no longer the option for ‚ÄúLegacy Packet Forwarder‚Äù.

                    Instead we set the Server Address like so‚Ä¶

                    1. Browse to the Gateway that we have added

                    2. Click ‚ÄúDownload global_conf.json‚Äù

                    3. Open the Downloaded global_conf.json with a text editor.

                      It should look like this‚Ä¶

                    4. On our RAKwireless Gateway, run this‚Ä¶

                    5. Select ‚ÄúEdit Packet Forwarder Config‚Äù

                    6. Look for the gateway_conf section‚Ä¶

                    7. Replace these values from the Downloaded global_conf.json‚Ä¶

  "gateway_ID":     ...,
  "server_address": ...,
  "serv_port_up":   ...,

                    9. Insert the entire servers section from the Downloaded global_conf.json‚Ä¶

  "gateway_ID":     ...,
  "server_address": ...,
  "serv_port_up":   ...,
  "serv_port_down": ...,

                    10. Save the file.
